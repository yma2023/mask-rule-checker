# C言語 基本文法まとめ（超入門テキスト）

この資料は、C言語を初めて体系的に学ぶ人向けに、「コード例＋短い理解」を積み重ねていく形式でまとめたものです。
一通り読めば、Cの基本構造が頭の中に一本の線でつながる構成になっています。

 

## Hello World から始める

C言語のプログラムは必ず `main` 関数から実行が始まります。
まずは「文字を表示するだけ」の最小構成を見てみます。

```c
#include <stdio.h>

int main(void){
    printf("Hello World\n");
    return 0;
}
```

`printf` を使うことで、画面に文字列を表示できます。
プログラムの入口が `main` である、というのが最初の重要ポイントです。

 

## printf関数と書式指定子

`printf` は、値を埋め込んで表示することもできます。

```c
int a = 10;
printf("a = %d\n", a);
printf("pi = %.2f\n", 3.14159);
```

`%d` は整数、`%f` は小数を表します。
`%.2f` のように書くと「小数点以下2桁まで表示」という意味になります。

 

## 演算の基本

数値の計算は、直感通りに書けます。

```c
int a = 7, b = 3;
printf("%d\n", a + b); // 足し算
printf("%d\n", a / b); // 整数除算
printf("%d\n", a % b); // 余り
```

ただし注意点として、整数同士の割り算は小数にならず、切り捨てになります。

 

## 変数という「箱」

変数は値を保存しておくための箱です。

```c
int x = 5;
x = x + 2;
```

一度代入した後でも、後から値を更新できます。
この「状態を持てる」という感覚が、プログラミングの基礎になります。

 

## データ型とキャスト

型が違う値を代入する場合、明示的に変換することがあります。

```c
double d = 3.8;
int i = (int)d;
```

これをキャストと呼び、小数点以下は切り捨てられます。

 

## 代入演算子の省略記法

次の2つは同じ意味です。

```c
x = x + 5;
x += 5;
```

後者は省略記法で、実務ではこちらがよく使われます。

 

## if文による分岐

条件によって処理を変えたいときに使います。

```c
if(x > 0){
    printf("正の数\n");
}
```

条件が真のときだけ、中の処理が実行されます。

 

## if–else / if–else if–else

条件が複数ある場合はこう書きます。

```c
if(x > 0){
    printf("正\n");
}else if(x < 0){
    printf("負\n");
}else{
    printf("ゼロ\n");
}
```

この構造で、ほとんどの分岐処理が書けます。

 

## 論理演算子（AND / OR）

条件を組み合わせることもできます。

```c
if(x > 0 && x < 10){
    printf("1〜9\n");
}
```

`&&` は AND、`||` は OR を意味します。

 

## switch文

値ごとに分岐する場合、`if` より見やすく書けることがあります。

```c
switch(x){
case 1: printf("1\n"); break;
case 2: printf("2\n"); break;
default: printf("その他\n");
}
```

数値や定数による分岐に向いています。

 

## for文（回数が決まっているループ）

```c
for(int i=0;i<5;i++){
    printf("%d\n", i);
}
```

回数があらかじめ分かっている処理に使います。

 

## 二重ループ

```c
for(int i=0;i<3;i++){
    for(int j=0;j<3;j++){
        printf("(%d,%d)\n", i, j);
    }
}
```

表・行列・座標処理などで頻繁に出てきます。

 

## while と do–while

```c
int i=0;
while(i<5){
    i++;
}
```

条件が真の間、繰り返します。

```c
int i=0;
do{
    i++;
}while(i<5);
```

こちらは「必ず1回は実行される」という違いがあります。

 

## 無限ループ

```c
while(1){
}
```

終了条件がないループです。
サーバー処理などでは使いますが、初心者は注意が必要です。

 

## 配列の基本

```c
int a[3] = {1,2,3};
printf("%d\n", a[0]);
```

同じ型のデータをまとめて管理できます。
インデックスは 0 から始まります。

 

## 文字列は char配列

```c
char s[] = "Hello";
printf("%s\n", s);
```

C言語では、文字列は「char型の配列」として扱われます。

 

## 関数

```c
int add(int a,int b){
    return a+b;
}
```

処理をひとまとまりにして再利用できるのが関数です。

 

## 変数のスコープ

```c
int g; // グローバル変数

void f(){
    int x; // ローカル変数
}
```

どこから参照できるか（有効範囲）が違います。
ここはバグの原因になりやすい重要ポイントです。

 

## 一言まとめ

C言語は本質的には、

* 順番に処理する
* 条件で分岐する
* 繰り返す
* データを構造として扱う

この4つを徹底的に鍛える言語です。
ここを理解できれば、PythonでもC++でも一気に楽になります。
